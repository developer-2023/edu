CSS Selectors

tag, #id, .class, []

ab - selects all elements that are both a and b (tag, class, id, data-...)

a b - selects all elements b that are descendants of the ancestor a

a > b - selects all elements b that are children of the parent a

a + b - selects the element b, which is the next neighbor of a

a ~ b - selects all elements b that are next neighbors of a

a, b - selects all elements that are a or b

* - selects all existing elements (universal selector)
